API PORT RUSSELL
Cette API permet de gérer les catways, les réservations et les utilisateurs du Port Russell.
https://api-russel-wv28.onrender.com/

Documentation API :
La documentation complète est disponible ici : https://api-russel-wv28.onrender.com/api-docs

Fonctionnalitéss :
- Gestion des catways (CRUD)
- Gestion des réservations (CRUD)
- Gestion des réservations par catways
- Authentification JWT
- Documentation Swagger

Outils :
- Node.js
- Express
- MongoDB
- Swagger UI
- JWT
